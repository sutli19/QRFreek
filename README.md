# QRFreek

A sleek and modern QR Code Generator built with React, Vite, and Tailwind CSS.

Generate QR codes instantly from URLs, text, emails, phone numbers, or any custom input and download them as PNG images—all within your browser.

---

## ✨ Features

- Instant QR Code Generation
- Download QR Codes as PNG
- Copy Input Content
- Clear Input Functionality
- Input Validation
- Responsive Design
- Dark / Light Theme Support
- Recent History using Local Storage
- Smooth Animations
- Glassmorphism-inspired UI
- Fast Client-Side Processing
- No Backend Required

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/sutli19/QRFreek.git
```

Navigate to project folder

```bash
cd qrfreek
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build production files

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
